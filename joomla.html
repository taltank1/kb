#Free CMS Platform: Joomla!


Before we begin our quest to explore another Popular Open Source CMS(Content Management System). First, lets try to understand the basics behind this popular technology, as we always do. The name “Joomla!” is a phonetic spelling for the Swahili word "Jumla," which means "all together" or "as a whole". Joomla! is a FREE! CMS, except it’s logo and the name. Developed by Open Source Matters, Inc. That’s a brief introduction & legal aspect of Joomla!. Now, lets answer What and Why?


What is Joomla! CMS?

Joomla! is a free and open-source content management system (CMS) for publishing web content. However, what is Joomla! as CMS actually means? CMS is a common term nowadays and we often confuse ourself between CMS and Framework, right?

A CMS is a software program which is specifically designed to to edit, add and manage websites which are not restricted to few pages and host large amount of content, videos, pictures, audio and what not. Moreover, you don’t need to have special coding knowledge and technical expertise in order to use it. Best examples of cms are Drupal, Joomla!, WordPress.

Framework, here means web application framework which is based on programming or coding. Being a developer you can choose any of the web application framework like Zend, Django, CakePHP, Struts, .NET, and even Rails.


Why and Who use Joomla! ?

It takes even less than fifteen minutes to download and ready script on server with Joomla!! Huge number of developers are working to improve it and that’s the reason it's very popular, stable, secure and easy to use. Few advantages hereunder would validate our point.

    ✔ Easy to use and install: Joomla! is an open source CMS and quite easy to install and simpler than Drupal.

    ✔ Joomla! For All: From single page portal to big corporate website rely upon Joomla!

    ✔ Active Support: Despite abundance of developers tools and tutorials are available, you can get extensive support on community forum.


Disadvantages

We will keep this short, since we have discussed all positive and rich features of Joomla!. Now, it feels like “The grass is greener on the other side of the fence”. However, it's not! Let us tell you why?
Many plugins, modules and extension are paid. Moreover, it gives you limited access for website customization and has less security level when compared to other CMS like Drupal.


Closing Thoughts

So as to conclude we can say every technology either CMS or Framework is built for a reason and it always depend on your project requirements to choose from the best suitable option. If you are looking for something more stable yet versatile CMS, which contains free script, handy plugins, modules, templates and good technical support then Joomla! Is ment for you.
